# 차량추가 양식 (루티_차량추가_양식.xlsx)

설정 > 차량 관리 > **차량 추가**(엑셀 업로드)에 사용하는 공식 양식의 필드 용어집입니다.
영문 Key는 엑셀 1행(시스템 컬럼명), 한글 필드명은 2행(화면 표시명) 기준입니다.

## 차량 기본 정보

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 차량 | `name` | 필수 | 차량 식별 정보 (예: 1호차, 12가1234, 5톤윙바디) |
| 아이디(휴대폰 번호) | `phone_number` | 필수 | 기사 휴대폰 번호. 입력한 번호로 기사 앱 초대장 전송 |
| 운영 유형 | `operation_type` | 필수 | 차량의 지입 여부 — 고정차 / 고정용차 / 지입차 |
| 차종 | `model_name` | 필수 | 차량 종류. 배차 시 차종에 따른 최적 경로 생성 |
| 연료 | `fuel_type` | 선택 | 경유 / 휘발유 / LPG 등 (유류비 계산에 사용) |
| 연비 | `fuel_efficiency` | 선택 | 차량 연비 (소수점 한 자리까지) |
| 메모 | `note` | 선택 | 차량에 대한 자유 메모 |

## 용적량 · 특수 조건

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 최대 용적량 | `max_capacity` | 필수 | 차량에 적재 가능한 최대 용적량 (정수) |
| 최대 용적량1 | `max_capacity1` | 선택 | 두 번째 기준 최대 용적량 |
| 최대 용적량2 | `max_capacity2` | 선택 | 세 번째 기준 최대 용적량 |
| 특수 조건 | `requirement` | 선택 | 차량이 처리 가능한 특수 조건 부여 (예: 냉장,1.4톤) |

## 근무 시간

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 주행 시작 시간 | `work_start_time` | 필수 | 차량이 실제 주행을 시작하는 시간 (예: 09:30) |
| 주행 종료 시간 | `work_end_time` | 선택 | 차량이 주행을 마치는 시간 |
| 휴게시간 (분) | `break_duration` | 선택 | 휴게시간 길이 (분 단위) |
| 휴게시간 구간 시작시간 | `break_start` | 선택 | 휴게시간을 배정할 시간 구간의 시작 |
| 휴게시간 구간 종료시간 | `break_end` | 선택 | 휴게시간을 배정할 시간 구간의 끝 |

## 출발지 · 도착지

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 출발지 주소 | `start_address` | 필수 | 주행을 시작할 출발지 도로명 주소 |
| 출발지 상세주소 | `start_detail_address` | 선택 | 출발지 상세주소 |
| 도착지 주소 | `end_address` | 선택 | 주행 종료 후 복귀할 도착지 도로명 주소 |
| 도착지 상세주소 | `end_detail_address` | 선택 | 도착지 상세주소 |

## 회전

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 회전 | `work_round` | 필수 | 출발지 기준 회전 수 (1~5). 1회전과 다회전 주문 처리 가능 |
| 회전 작업시간 (분) | `work_round_time` | 선택 | 출발지에서 다음 회전까지 소요되는 작업 시간 (10~180분) |
