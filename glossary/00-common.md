# 공통 용어

ROOUTY TMS 전 메뉴에서 공통으로 사용되는 조회·표·내비게이션 용어입니다.

## 전역 내비게이션 (GNB)

| 용어 | English | 정의 |
|---|---|---|
| 주문관리 | Order Management | 주문의 등록, 조회, 수정, 취소 등 주문 전반을 관리하는 메뉴 |
| 배차 계획 | Dispatch Planning | 배차 옵션을 설정하고 주문과 차량을 선택하여 배차를 계획하는 메뉴 |
| 모니터링 | Monitoring | 배차 목록을 조회하고 주행 중인 배차 주문을 실시간 모니터링하는 메뉴 |
| 보고서 | Report | 전체·차량별 주행 기록을 관리하고 현황을 확인하는 메뉴 |
| 정산 | Settlement | 운송 비용·정산 관리 메뉴 (현재 비활성 · 준비 중) |
| 메시지 관리 | Message Management | 고객·기사·거래처에게 문자 메시지를 발송·예약·자동화하는 메뉴 |
| 설정 | Settings | 조직·리소스·기준 정보·PoD를 관리하는 메뉴 (톱니바퀴 아이콘) |
| 사용방법 문의 | Support Contact | 고객 지원 연락처 (1533-0441) |
| 공지사항 | Notice | 서비스 공지 페이지 |
| FAQ | FAQ | 자주 묻는 질문 페이지 |

## 조회 · 필터 공통

| 용어 | English | 정의 |
|---|---|---|
| 조회하기 | Search | 설정한 조건으로 목록을 검색 |
| 초기화 | Reset | 검색 조건을 기본값으로 되돌림 |
| 조회 기간 | Search Period | 조회 기준 날짜 범위 (오늘/1주일/1개월/3개월 또는 직접 지정) |
| 조회 항목 | Search Results Count | 검색 결과로 조회된 건수 |
| N개씩 보기 | Rows per Page | 한 페이지에 표시할 행 수 (10/20/30/40/50/300/500) |
| 다운로드 | Download | 조회된 목록을 파일로 내려받기 |
| 표 항목 조회 설정 | Table Column Settings | 목록에 표시할 컬럼을 노출/미노출로 관리하는 설정 (톱니바퀴) |
| 노출 목록 | Visible Columns | 테이블에 표시되는 컬럼 목록 |
| 미노출 목록 | Hidden Columns | 테이블에 표시되지 않는 컬럼 목록 |
| 소속 팀 | Team | 데이터(차량·주문·매니저 등)가 속한 팀 |

## 핵심 도메인 개념

| 용어 | English | 정의 |
|---|---|---|
| 배차 | Dispatch | 주문을 차량(기사)에 배정하는 행위 |
| 자동 배차 | Auto Dispatch | 최적화 알고리즘으로 주문을 차량에 자동 배정하는 방식 |
| 자가 배송 | Self Delivery | 배차 시스템을 거치지 않고 화주가 직접 배송을 수행하는 방식 |
| 주행 | Route / Trip | 배차 확정 후 차량이 경로를 따라 운행하는 것 |
| 경로ID | Route ID | 배차(주행) 건에 시스템이 부여하는 고유 식별자 |
| 회전 | Trip / Round | 차량이 출발지에서 출발해 배송 후 복귀하는 1회 운행 단위 |
| 다회전 | Multi Round | 한 차량이 출발지 기준으로 1회전과 여러 회전 주문 처리를 하는 방식 |
| 밀크런 | Milk Run | 한 차량이 여러 지점을 순회하며 수거·배송을 함께 수행하는 운송 방식 |
| 권역 | Zone / Region | 배차 시 차량에게 담당시킬 지역 단위 |
| 용적량 | Capacity | 주문·차량의 적재량 지표. 무게/부피/수량 등 최대 3개 기준(용적량1~3)으로 관리 |
| 특수 조건 | Special Condition | 주문과 차량을 매칭하는 조건 태그 (예: 냉장, 1.4톤). 조건이 걸린 주문은 같은 조건의 차량에만 배차됨 |
| 화주사 | Shipper | 배송 처리를 의뢰한 제품의 소유자·의뢰 업체 |
| 중개사 | Broker | 주문을 담당하는 영업사원·대리인 등 중개 주체 |
| 납품처 | Delivery Point | 반복 배송하는 고정 배송지 (설정에서 사전 등록) |
| PoD | Proof of Delivery | 작업 완료 증빙 (인수증·작업 완료 사진 등) |
