# 주문등록 양식 (루티_주문등록_양식.xlsx)

주문관리 > 주문 추가 > **여러 건 엑셀로 추가**에 사용하는 공식 업로드 양식의 필드 용어집입니다.
영문 Key는 엑셀 1행(시스템 컬럼명), 한글 필드명은 2행(화면 표시명) 기준입니다.

## 기본 정보

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 업체 주문 번호 | `client_order_key` | 선택 | 귀사에서 사용 중인 고유 주문번호 |
| 주문 접수일 | `received_date` | 필수 | 주문이 귀사에 접수된 날짜 (미래 날짜 입력 불가) |
| 주문 유형 | `service_type` | 선택 | 배송/수거 구분. 비어있으면 기본값 배송 |
| 배차 우선순위 | `priority` | 선택 | 높음/보통/낮음 — 우선 배차되어야 하는 주문 지정 |

## 고객 정보

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 고객명 | `name` | 필수 | 고객의 성함 또는 상호 |
| 고객 연락처 | `contact` | 선택 | 고객 전화번호 (예: 021234567, 010-0000-0000) |
| 주소 | `address` | 필수 | 도로명 주소와 건물번호 (주소정보누리집 기준 검증) |
| 상세주소 | `detail_address` | 선택 | 상세 주소 |
| 고객 전달사항 | `note` | 선택 | 기사에게 전달할 고객 요청사항 |

## 용적 · 아이템 정보

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 합산 용적량1 | `capacity` | 필수 | 하나의 주문에 대한 총 용적량 (정수) |
| 합산 용적량2 | `capacity1` | 선택 | 두 번째 기준 용적량 |
| 합산 용적량3 | `capacity2` | 선택 | 세 번째 기준 용적량 |
| 아이템명 | `product_name` | 선택 | 상품명 |
| 아이템 코드 | `product_code` | 선택 | 귀사에서 사용 중인 상품코드 |
| 아이템 수량 | `product_quantity` | 선택 | 상품 수량. 비어있으면 기본값 1 |

## 거래처 정보

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 화주사명 | `consignor_name` | 선택 | 배송 처리를 의뢰한 제품의 소유자·의뢰 업체 |
| 화주사 연락처 | `consignor_contact` | 선택 | 화주사 전화번호 |
| 중개사명 | `salesman_name` | 선택 | 주문을 담당하는 영업사원·대리인 등 |
| 중개사 연락처 | `salesman_contact` | 선택 | 중개사 전화번호 |

## 작업 조건

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 밀크런 | `milk_run_id` | 선택 | 수거한 상품을 특정 주소에 배송할 때 사용하는 그룹 ID (영어·숫자, 예: milkrun7, PnD) |
| 예상 작업 시간(분) | `service_duration` | 선택 | 도착 후 작업 예상 시간 (정수) |
| 담당 차량 지정 | `service_vehicle` | 선택 | 주문을 담당할 차량 지정 ([설정 > 차량 관리] 등록 차량) |
| 특수 조건 | `requirement` | 선택 | 특수 조건 차량이 처리해야 할 때 지정 (예: 냉장,1.4톤) |
| 작업 희망일 | `desired_date` | 선택 | 배송/수거 희망 날짜 (과거 날짜 입력 불가) |
| 희망 시간(이후) | `desired_after_time` | 선택 | 도착 희망 시간 범위의 시작 (예: 15:30) |
| 희망 시간(이전) | `desired_before_time` | 선택 | 도착 희망 시간 범위의 끝 (예: 18:00) |

## 비고

| 필드명 | Key | 필수 | 정의 |
|---|---|---|---|
| 비고1~5 | `customer_field_1`~`customer_field_5` | 선택 | 주문에 대한 자유 메모 (예: 현관 비밀번호, 배송코드) |

> 참고: 양식 파일에는 배차관리 경로 다운로드 양식(구버전, 미사용) 시트가 포함되어 있으나 현재 업로드에는 `주문 업로드` 시트만 사용합니다.
