# 설정

조직·리소스·기준 정보·PoD를 관리하는 메뉴입니다.

> 기준 화면: `tms.roouty.io/setting`

## 설정 메뉴 구조

| 분류 | 하위 메뉴 |
|---|---|
| 조직 관리 | 팀 관리 · 매니저 관리 |
| 리소스 관리 | 차량 관리 · 권역 관리 · 앱 관리 |
| 기준 정보 관리 | 납품처 관리 · 특수 조건 관리 · 팔레트 용적량 관리 · 주문 분할 관리 |
| PoD관리 | 인수증 관리 · 작업 보고 관리 |

## 팀 관리

| 용어 | English | 정의 |
|---|---|---|
| 팀 | Team | 차량·매니저·주문을 묶는 조직 단위 |
| 팀 추가 | Add Team | 신규 팀 등록 |
| 소속 매니저 / 소속 차량 | Team Managers / Vehicles | 팀에 속한 매니저 수·차량 대수 |
| 추가 일자 | Created Date | 팀 등록 일자 |
| 팀 연락처 / 팀 주소 / 팀 상세주소 / 팀 메모 | Team Contact / Address / Detail / Memo | 팀 기본 정보 |
| 하위 구분 | Sub-category | 팀에 연결된 하위 구분 값 |

## 차량 관리

| 용어 | English | 정의 |
|---|---|---|
| 일반 차량 / 기타 차량 | Regular / Other Vehicles | 차량 목록 탭 구분 |
| 차량 추가 | Add Vehicle | 신규 차량 등록 (직접 입력 또는 차량추가 양식 업로드) |
| 설치 문자 전송 | Send Install SMS | 기사 앱 설치 안내 문자를 차량(기사)에게 발송 |
| 차량 일괄 수정 | Bulk Edit Vehicles | 선택한 여러 차량 정보를 한 번에 수정 |
| 차량 삭제 | Delete Vehicle | 선택한 차량 삭제 |
| 활성 상태 | Active Status | 차량 사용 여부 (활성/비활성) |
| 차량 연락처 | Vehicle Contact | 기사 휴대폰 번호 (기사 앱 계정) |
| 운영 유형 | Operation Type | 고정차 / 지입차 / 고정용차 / 용차 |
| 고정차 | Fixed Vehicle | 자사가 직접 운영하는 차량 |
| 지입차 | Owner-operated Vehicle | 지입 계약으로 운영되는 차량 |
| 고정용차 | Fixed Chartered Vehicle | 고정 계약된 용차 |
| 용차 | Chartered Vehicle | 필요 시 일시 계약하는 외부 차량 |
| 등록 일자 | Registered Date | 차량 등록 일자 |
| 앱 설치 | App Installed | 기사 앱 설치 여부 |
| 스마트 주행 | Smart Driving | 기사 앱 스마트 주행 기능 사용 여부 |
| 차종 | Vehicle Model | 차량 종류 (배차 시 차종에 따른 최적 경로 생성) |
| 연료 / 연비 | Fuel Type / Efficiency | 유류비 계산에 사용되는 연료 종류(경유/휘발유/LPG)와 연비 |
| 용적량1~3 | Capacity 1–3 | 차량에 적재 가능한 최대 용적량 |
| 회전 / 회전 작업 시간 | Rounds / Round Work Time | 차량의 회전 수(1~5)와 회전 간 작업 시간(10~180분) |
| 담당 권역 | Assigned Zone | 차량이 담당하는 권역 |
| 근무 시작/종료 시간 | Work Start / End Time | 차량의 주행 가능 시간대 |
| 휴게 시간 | Break Time | 차량의 휴게시간 (분) |
| 출발지/도착지 주소 | Start / End Address | 주행 출발지와 복귀 도착지 |
