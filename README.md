# ROOUTY Solution Glossary

Official glossary of terms used in ROOUTY, WeMeet Mobility's route optimization solution.

위밋모빌리티 경로 최적화 솔루션 **ROOUTY(루티)**의 공식 용어집입니다.
ROOUTY TMS의 전체 메뉴·탭 화면과 공식 등록 양식(엑셀)을 기준으로 용어를 수집·정리했습니다.

> 📌 수집 기준: `tms.roouty.io` 전체 메뉴 · 루티 주문등록/차량추가 양식 · 최종 수집일: 2026-07-16

## 처음 오셨나요?

1. **[공통 용어](glossary/00-common.md)** — 배차·주행·권역·용적량 등 루티의 핵심 개념부터 익히세요.
2. **[주문관리](glossary/01-order-management.md)** → **[배차 계획](glossary/02-dispatch-planning.md)** → **[모니터링](glossary/03-monitoring.md)** — 실제 업무 흐름(주문 등록 → 배차 → 관제) 순서입니다.
3. 고객 응대·엑셀 업로드 업무를 맡는다면 **[등록 양식 용어집](forms/index.md)**을 함께 보세요.

각 문서에는 실제 화면 캡처가 포함되어 있어, 용어가 화면 어디에 있는지 바로 확인할 수 있습니다.
상단 검색창에서 용어를 바로 검색할 수도 있습니다.

> ⚠️ 정산 메뉴는 현재 비활성(준비 중) 상태로 용어 수집 대상에서 제외했습니다.

## 표기 규칙

- 메뉴 용어는 `구분 | 용어 | English | 정의` 4열 표로 정리합니다.
- **구분** 열은 해당 용어가 화면에서 쓰이는 위치를 나타냅니다:
  - `메뉴` 상단/좌측 내비게이션 · `탭` 화면 내 탭 · `버튼` 클릭해서 실행하는 기능 · `필드` 입력/표시 항목 · `필터` 검색 조건 · `상태` 상태값 · `옵션` 선택지
- 등록 양식 필드는 `필드명(한글) | Key(영문) | 필수 여부 | 정의` 4열 표로 정리합니다.
- 화면에 노출되는 워딩을 그대로 표제어로 사용합니다.
- ⓘ 아이콘에 마우스를 올리면 나오는 도움말은 💬 표시와 함께 정의에 포함했습니다.
